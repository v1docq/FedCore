Installation
------------
To install the package an MPI should be installed.
For Linux it can be done with the following commands:

.. code-block::

    sudo apt-get update
    sudo apt-get install -y openmpi-bin libopenmpi-dev


Then using PIP:

.. code-block::

    pip install fedcore
